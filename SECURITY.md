# セキュリティガイドライン

## APIキーの安全な管理

### ✅ 推奨方法

1. **環境変数ファイル (.env)**
   ```
   OPENAI_API_KEY=your_actual_api_key_here
   ```

2. **システム環境変数**
   - Windows: 環境変数に`OPENAI_API_KEY`を追加

### ❌ 避けるべき方法

- APIキーをコード内に直接記述
- APIキーをGitにコミット
- APIキーを公開リポジトリにアップロード
- APIキーをログに出力

### 🔒 セキュリティチェック

- [ ] `.env`ファイルが`.gitignore`に含まれている
- [ ] APIキーがコード内に直接記述されていない
- [ ] 環境変数が正しく設定されている
- [ ] APIキーが公開されていない

### 📝 設定手順

1. `.env`ファイルを作成
2. 実際のAPIキーを設定
3. アプリケーションを再起動
4. 動作確認
